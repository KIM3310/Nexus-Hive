Thank you for installing {{ .Chart.Name }} v{{ .Chart.Version }} (appVersion {{ .Chart.AppVersion }}).

Release:   {{ .Release.Name }}
Namespace: {{ .Release.Namespace }}

1. Check the rollout status:

   kubectl -n {{ .Release.Namespace }} rollout status deployment/{{ include "nexus-hive.fullname" . }} --timeout=180s

2. Verify the health endpoint (from inside the cluster):

   kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "nexus-hive.fullname" . }} 8000:{{ .Values.service.port }}
   curl http://localhost:8000/health

3. Run the gold evaluation suite:

   curl http://localhost:8000/api/evals/nl2sql-gold/run

{{- if .Values.ingress.enabled }}

4. Ingress is enabled. The service is exposed at:
{{- range .Values.ingress.hosts }}
     https://{{ .host }}
{{- end }}
{{- else }}

4. Ingress is disabled. Expose the service with:

   kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "nexus-hive.fullname" . }} 8000:{{ .Values.service.port }}
{{- end }}

{{- if .Values.autoscaling.enabled }}

5. HorizontalPodAutoscaler is enabled:
     min: {{ .Values.autoscaling.minReplicas }}
     max: {{ .Values.autoscaling.maxReplicas }}
     target CPU: {{ .Values.autoscaling.targetCPUUtilizationPercentage }}%

   kubectl -n {{ .Release.Namespace }} get hpa {{ include "nexus-hive.fullname" . }}
{{- end }}

{{- if .Values.ollama.enabled }}

6. Ollama sidecar is enabled. Pull the default model after install:

   kubectl -n {{ .Release.Namespace }} exec deploy/{{ include "nexus-hive.fullname" . }} -c ollama -- ollama pull {{ .Values.ollama.model }}
{{- end }}

{{- if .Values.serviceMonitor.enabled }}

7. ServiceMonitor is enabled. Verify scraping from Prometheus:

   kubectl -n {{ .Release.Namespace }} get servicemonitor {{ include "nexus-hive.fullname" . }}
{{- end }}
